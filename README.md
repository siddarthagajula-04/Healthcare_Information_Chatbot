# 🏥 Healthcare Information Chatbot

## 📌 Overview

A domain-specific chatbot that provides general healthcare information using the Google Gemini API. The application is built with Streamlit and follows a modular backend architecture suitable for cloud deployment.

## 🚀 Features

- Healthcare-related information support
- Multi-turn conversation memory
- Gemini API integration
- Streamlit chat interface
- AWS EC2 deployment

## 🧰 Tech Stack

Python • Streamlit • Google Gemini API • AWS EC2 • python-dotenv

## 🏗️ Architecture

User → Streamlit UI → Backend → Prompt Module → Gemini API → Response

## ▶️ Run Locally

- pip install -r requirements.txt
- streamlit run app.py

## ☁️ Deployment

Deployed on AWS EC2 with port 8501 exposed for public access.
